# Smart Temperature & Humidity Monitoring System

## Overview
This project monitors temperature and humidity using a DHT22 sensor. The readings are displayed on a 16×2 I2C LCD. If the temperature exceeds a predefined threshold, a buzzer is activated to provide an alert.

## Components
- Arduino Uno
- DHT22 Sensor
- 16×2 I2C LCD
- Buzzer
- Jumper Wires

## Software
- Arduino IDE
- Proteus (for simulation)

## Features
- Real-time temperature monitoring
- Real-time humidity monitoring
- LCD display output
- High-temperature buzzer alert
